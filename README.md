# 🎓 Sistem Informasi Akademik XYZ

Project Ujian Akhir Semester (UAS) Mata Kuliah Pemrograman Web.

## 📌 Deskripsi

Sistem Informasi Akademik XYZ merupakan aplikasi berbasis PHP Native yang digunakan untuk mengelola data akademik seperti mahasiswa, dosen, mata kuliah, KRS, dan nilai.

---

## 🚀 Teknologi yang Digunakan

- PHP Native
- MySQL / MariaDB
- Bootstrap 5
- jQuery
- AJAX
- Font Awesome

---

## ✨ Fitur

- Login Administrator
- Dashboard
- CRUD Mahasiswa
- CRUD Dosen
- CRUD Mata Kuliah
- CRUD KRS
- CRUD Nilai
- Validasi Data
- Bootstrap Alert
- Live Date & Time
- AJAX Validasi
